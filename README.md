# Hola 👋🏻, soy Guillermo 💻📚

Soy un Desarrollador Web enfocado en crear soluciones eficientes y escalables. Mi camino hacia la tecnología comenzó de forma autodidacta y se consolidó con mi Diplomado en Desarrollo de Soluciones Web en la Universidad Central de Venezuela (UCV). 

Tengo un enfoque **fundamentalista** en el desarrollo: no me limito a usar herramientas o frameworks porque estén de moda, sino que busco entender el *porqué* detrás de cada tecnología. Creo firmemente que las bases sólidas y las buenas prácticas de arquitectura de software son la clave para un crecimiento profesional real.

## ⚙️ Metodologías y Flujo de Trabajo

* **Desarrollo Asistido por IA:** Capacidad para orquestar flujos de trabajo con inteligencia artificial, estructurando archivos de contexto técnicos y precisos para optimizar la generación y revisión de código.
* **TDD (Test-Driven Development):** Enfoque guiado por pruebas para garantizar que el código (ya sea escrito manualmente o co-creado con IA) cumpla estrictamente con la lógica de negocio y los estándares de calidad antes de su implementación.

## 🧠 Filosofía y Habilidades Complementarias

* **Investigación y Optimización:** Disfruto encontrar la forma más óptima de resolver un problema técnico, yendo siempre a la documentación oficial y entendiendo las bases teóricas.
